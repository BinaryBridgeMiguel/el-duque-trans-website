## Customization

### Phone Numbers & Contact
Update the placeholder phone numbers (`+32 123 456 789`) and email addresses with actual contact info.

### Form Integration
The contact form is set up for Formspree. Update the form action URL:
```html
<form action="https://formspree.io/f/YOUR-FORM-ID" method="POST">
```

### Images
Replace placeholder images from Unsplash with actual company photos.
